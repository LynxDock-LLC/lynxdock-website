# Security Policy

## Reporting a vulnerability

Email **security@lynxdock.app**. Please do **not** open a public issue or
disclose the vulnerability publicly before we have confirmed and addressed it.

We follow responsible disclosure: we will acknowledge your report as quickly as
we can, keep you updated on remediation, and (if you wish) credit you once a fix
is released.

## Supported versions

LynxDock is early-stage. Security fixes target the latest `main` of each
repository.
